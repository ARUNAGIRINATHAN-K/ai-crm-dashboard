# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-04-20

### Added

- Initial release — first 40+ curated entries across all three layers (Skills, Workflows, Environments)
- Agent skills library covering perception, memory, planning, execution, and communication
- Real-world workflow guides for research, code generation, data pipelines, and browser automation
- Secure execution environments section with hosted sandboxes, VMs, containers, and serverless runtimes
- Frameworks and orchestration comparison table
- Four agentic pattern deep-dives: ReAct, plan-and-execute, reflection loop, multi-agent
- Contributing guide with tag taxonomy and acceptance criteria
- GitHub Actions workflow for weekly dead link checking
- PR template and issue template for new entry requests
- MIT license
